DFD Level 1 Details

1. Data Stores

Store IDNameUsage & Purpose
D1User DBAuthentication credentials, user roles (customer/admin), and profile information.
D2Book DBMaster catalog containing book titles, ISBNs, prices, and current stock_quantity.
D3Order/Invoice DBImmutable record of historical transactions, status (Pending/Shipped), and snapshots of items.

2. Processes

Process IDTaskFunctional Description
1.0Login/AuthVerifies user credentials against D1 and manages JWT session tokens.
2.0Browse & SearchProvides read-only access to the book catalog in D2 based on user queries.
3.0Place OrderCoordinates checkout: validates stock in D2, creates order in D3, and clears cart.
4.0Stock MgmtInternal admin tool for replenishing or adjusting inventory levels in D2.
5.0ReportingAggregator process that pulls data from D3 to generate sales performance metrics for Admins.

3. Significant Data Flows

  • User → P1.0: Login credentials (Email/Password).
  • P3.0 → D2: Update stock_quantity (decrements inventory upon successful order).
  • D3 → P5.0: Raw transaction data for business intelligence.
  • P5.0 → Admin: Formatted sales reports and order summaries.
